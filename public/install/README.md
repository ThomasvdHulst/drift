# Install-guide screenshots

Drop the iOS screenshots here with these exact names and the `/install` page picks
them up automatically (until then it shows a labelled placeholder in each slot):

| File | What it should show |
|---|---|
| `ios-1-share.png` | Safari open on usedrift.org, with the Share button (square + up-arrow) in the toolbar |
| `ios-2-add.png` | The share sheet scrolled to the "Add to Home Screen" row |
| `ios-3-confirm.png` | The "Add to Home Screen" confirm screen with the Drift name/icon and the Add button |

Tips:
- Portrait phone screenshots (the page shows them at ~200px wide, tall aspect).
- PNG. Crop to just the phone screen (no need for a device frame).
- Android needs no screenshots (that section is text only).
